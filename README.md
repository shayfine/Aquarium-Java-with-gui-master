# Aquarium-Java-with-gui

######  Aquarium with gui in java with multi threads.
Swimmable animals try to eat the worms.
**Design patterns implements in this project:
**
1. Singleton 
2. Abstract Factory
3. Prototype .C
4. Listener/Observer 
5. Decorator 
6. Memento
